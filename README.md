# HR-Analytics-Dashboard
Interactive HR Analytics Dashboard using Tableau


## Project Overview
This Tableau dashboard analyzes employee attrition, job satisfaction, gender distribution, and department-wise employee trends.

## Tools Used
- Tableau
- Excel
- Data Visualization

## Key Insights
- Attrition Rate Analysis
- Department-wise Attrition
- Gender Analysis
- Employee Age Group Trends
- Job Satisfaction Ratings
